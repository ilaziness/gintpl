## <center>GinTpl</center>

基于Gin框架的快速开发脚手架

Rapid development scaffolding based on Gin framework

## 目录结构

```shell
├─cmd                   // main入口
├─config                // 配置文件
├─internal              // 应用私有包
│  ├─app                // 实际应用目录
│  │  └─web             // web应用示例
│  ├─dao                // ent dao示例
│  ├─ent                // ent示例
│  ├─errcode            // 应用公用错误码示例
│  ├─queue              // 队列使用示例
│  └─timer              // 定时器使用示例

```
